---
title: "Verify Token"
api: "GET /auth/verify"
description: "Verify the validity of an authentication token"
---

## Request

<ParamField header="Authorization" type="string" required>
  The authentication token to verify
</ParamField>

## Response

### Success Response

Returns HTTP status `200` with no body when the token is valid.

## Error Responses

<ResponseField name="error" type="string">
  Error message describing what went wrong
</ResponseField>

### 401 Unauthorized

Returned when no token is provided in the Authorization header.

```json
{
  "error": "Unauthorized: No token provided"
}
```

### 403 Forbidden

Returns HTTP status `403` with no body when the token is invalid or expired.

### 500 Internal Server Error

Returned when an unexpected error occurs during token verification.

```json
{
  "error": "An unexpected error occurred. Please try again later."
}
```

## Examples

<CodeGroup>

```bash cURL
curl -X GET https://api.planttogether.com/auth/verify \
  -H "Authorization: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript
const response = await fetch('https://api.planttogether.com/auth/verify', {
  method: 'GET',
  headers: {
    'Authorization': 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
  }
});

if (response.ok) {
  console.log('Token is valid');
} else if (response.status === 403) {
  console.log('Token is invalid or expired');
} else if (response.status === 401) {
  console.log('No token provided');
}
```

```python Python
import requests

response = requests.get(
    'https://api.planttogether.com/auth/verify',
    headers={
        'Authorization': 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
    }
)

if response.status_code == 200:
    print('Token is valid')
elif response.status_code == 403:
    print('Token is invalid or expired')
elif response.status_code == 401:
    print('No token provided')
```

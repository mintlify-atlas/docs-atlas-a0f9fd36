---
title: "Log In"
api: "POST /auth/login"
description: "Authenticate an existing user with email and password"
---

## Request

<ParamField body="email" type="string" required>
  The email address of the user account
</ParamField>

<ParamField body="password" type="string" required>
  The password for the user account
</ParamField>

## Response

<ResponseField name="token" type="string">
  Authentication token for the authenticated user
</ResponseField>

### Success Response

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

## Error Responses

<ResponseField name="error" type="string">
  Error message describing what went wrong
</ResponseField>

### 500 Internal Server Error

Returned when an unexpected error occurs during authentication, including invalid credentials.

```json
{
  "error": "An unexpected error occurred. Please try again later."
}
```

## Examples

<CodeGroup>

```bash cURL
curl -X POST https://api.planttogether.com/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john@example.com",
    "password": "securePassword123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.planttogether.com/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'john@example.com',
    password: 'securePassword123'
  })
});

const data = await response.json();
console.log(data.token);
```

```python Python
import requests

response = requests.post(
    'https://api.planttogether.com/auth/login',
    json={
        'email': 'john@example.com',
        'password': 'securePassword123'
    }
)

data = response.json()
print(data['token'])
```
